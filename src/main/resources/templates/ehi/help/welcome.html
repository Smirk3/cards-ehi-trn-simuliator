Getting started
1. Configuration
  • Go to settings page and click "Use default settings" button.
  • Check application variables
  • Go to list of cards page and add cards you want to do request for.
Do requests
  • Fill in main request parameters on messages page and click "Next" button.
  • You will see automatically generated XML request body. You can change this XML document manually by clicking "Edit" button on the top right corner of XML document frame. and then click "Save" button.
  • Click "Submit" button on the bottom of XML document
